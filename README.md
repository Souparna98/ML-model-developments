# ML-model-developments
ML models using Azure Databricks
 - This repo provides a practical guide to developing traditional machine learning models on Databricks and workflows using popular ML libraries. 
 - Additionally, the repo covers MLflow for tracking & managing model training for streamlined and effective model development in the Databricks environment.
- Databricks Runtime version: **17.3.x-cpu-ml-scala2.13** was used for running all notebooks.
